# Zenist

[English](README.md) | 繁體中文 | [简体中文](README_zh-CN.md)

---

**Zenist** 是一款專注於極簡的待辦事項管理工具。

Zenist 的設計核心建立於以下三大理念：
* **簡單**：純粹、無干擾的介面設計。
* **無推送**：不打擾您的注意力，拒絕任何非必要的通知。
* **透過自己的雲端硬碟同步**：所有數據皆透過您個人的雲端硬碟（Dropbox）進行同步，確保資料的絕對隱私與所有權。

### 功能特點
* **極簡介面**：採用黑白灰的純粹視覺，讓您專注於當下的任務。
* **背景執行**：支援常駐系統匣，隨時喚醒。
* **單一實例**：確保系統資源的高效利用，防止重複啟動。
* **離線優先**：本機端資料庫提供極致流暢的體驗，並在背景安靜地為您同步。
